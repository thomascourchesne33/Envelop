FC&F Modèles — Instructions d'installation
==========================================

1. Copiez FCF_Modeles.exe dans un emplacement permanent
   (ex. : C:\Program Files\FCFModeles\FCF_Modeles.exe)

2. Double-cliquez sur FCF_Modeles.exe pour lancer l'application

3. Au premier lancement, sélectionnez le fichier .xlsx des modèles
   situé dans votre dossier OneDrive partagé.

4. L'application s'installe automatiquement au démarrage de Windows
   après la première configuration.

5. Un icône apparaît dans la barre des tâches (zone de notification).
   Clic droit sur l'icône pour accéder aux paramètres.

UTILISATION
-----------
- Dans Outlook, ouvrez ou répondez à un courriel
- Tapez un code de modèle (ex. : /renou) dans le corps du message
- Une fenêtre flottante apparaît avec les modèles correspondants
- Sélectionnez un modèle, remplissez les champs variables, cliquez Insérer

PRÉREQUIS
---------
- Windows 10 ou Windows 11
- Microsoft Outlook (version bureau classique)
- Aucune installation Python requise

DÉPANNAGE
---------
- Journal d'erreurs : %APPDATA%\FCFModeles\error.log
- Si Outlook n'est pas détecté, assurez-vous qu'un courriel est ouvert
  en mode composition ou réponse.
